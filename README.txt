SheetPress Connects Google sheets with WordPress to manage category, tags and Yoast Meta to make things easy.

== Description ==

Managing metadata like categories, tags, and SEO (by Yoast) is really hard if we go edit each post in its edit Window. Here is where SheetPress makes things easy by connecting your pages and posts with Google sheet from where you can easily manage and update metadata in no time.


[youtube https://www.youtube.com/watch?v=Mwbw0Qlfv24]


#### How SheetPress Can make things easy?

* You can manage Yoast SEO data for Posts with Google sheets and sync it back to WordPress in one click.
* Easily Manage Post Categories and tags with Google sheets
* You can manage Yoast SEO data for Pages with Google sheets and sync it back to WordPress in one click.

#### Documentation
For help setting up and configuring SheetPress please refer to our [user guide](http://wpgeniuz.com/docs/docs/sheetpress/) 

== Installation ==

1. Search `SheetPress` in Add new Plugin window in your dashboard
2. Install and activate the plugin
3. Connect your Google sheets from settings page and you are good to go
